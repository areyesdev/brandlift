# Website BrandLift

### 1. Install Dependencies

```bash
npm install
# or
yarn install
# or (recommended)
pnpm install
```

### 2. Start development Server

```bash
npm run dev
# or
yarn dev
# or (recommended)
pnpm dev
```

### Preview & Build

```bash
npm run preview
npm run build
# or
yarn preview
yarn build
# or (recommended)
pnpm preview
pnpm build
```
